程式可處理：
多個反應物與多個生成物
任意化學計量係數
已知平衡常數 𝐾的化學反應
使用者只需輸入各物種的化學計量係數、初始濃度與平衡常數，程式即可數值求解反應的平衡組成。

四、開發過程說明:
在計算平衡濃度時，發現只要反應變得稍微複雜一點，手動計算平衡濃度就會花很多時間，也很容易算錯，因此產生了想用程式來幫忙計算的想法。
開發初期因為教授說太簡單並思考實際使用時會遇到的需求，決定讓程式能夠輸入多個反應物與生成物，並自動計算平衡狀態下的濃度。
在實際撰寫程式的過程中，遇到不少具體的困難。一開始最常出現的問題是計算結果不合理，例如反應物的平衡濃度被算成負值，或是生成物濃度超大。檢查後才發現是反應進行量 
x 的範圍沒有設定好，導致程式在計算時超出了實際可發生的反應程度，後來改成以上界限制反應物不被完全消耗，才解決這個問題。另一個困難是數值解法不穩定，有時候程式會一直跑得不到結果，
或每次算出來的答案差異很大，最後透過調整二分法的判斷條件，才讓計算結果穩定。在撰寫平衡函數時，也因為次方與乘積順序寫錯，導致計算出的反應商與理論值不符，只能逐步print中間結果來找出錯誤。
